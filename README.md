# Path-IM-Server-OICQ
【一个demo】打开了oicq   聊天记录停步去年的深秋

> 此项目基于[Path-IM-Server](https://github.com/Path-IM/Path-IM-Server)构建，提供了一个简单的IM服务器。

> 开发过程请查阅[README.md](docs/dev/README.md)文件。

# 功能列表
## 用户模块
### 无需鉴权
- 用户名是否存在
- 用户注册
- 用户登录
### 需要鉴权
- 用户信息查询(批量)
- 用户信息修改
- 用户配置(kv存储)查询
- 用户配置修改
## 群组模块
- 群组创建
- 群组信息查询
- 群组信息修改
- 解散群组
- 群组成员查询
- 群组成员增加
- 群组成员删除
- 群组成员会话设置
## 关系
### 好友
- 好友列表查询
- 查询用户间的好友关系
- 查询和好友的会话选项 备注
- 修改和好友的会话选项 备注
- 好友添加
- 好友删除
### 黑名单
- 黑名单列表查询
- 查询用户间的拉黑关系
- 黑名单添加
- 黑名单删除
